# 朝読み（asayomi）

毎朝 [ovice](https://www.ovice.com/) で行うオンライン読書会の進行支援ツール（フェーズ1: 進行役1画面 + 画面共有）。

## 使い方
1. `index.html` を**ブラウザにドラッグ＆ドロップ**、またはダブルクリックして開く（サーバ不要）。
2. ① 今日読む範囲をスライダーで設定 →「黙読スタート」
3. ② 黙読タイマー（既定15分）
4. ③ 感想シェア（順番表示・クリックで話者選択・各1分・リアクション集計）
5. ④ サマリー（進捗バー・連続日数・明日の一歩）→「保存して終了」

- 操作はタップ/スライダー中心。キーボード入力は本の総ページ登録など最小限。
- データは localStorage に保存。⚙︎ から JSON エクスポート/インポート可。

## 別のPCで使う（ファイルコピー）
このアプリは外部サーバや通信に依存しないため、**フォルダごとコピーするだけ**で別PCでも動きます。

1. この `asayomi` フォルダを丸ごと別PCにコピー（USB / クラウドドライブ / GitHub など）。
2. コピー先で `index.html` を開く（ダブルクリック）。

### データ（進捗・履歴・参加者）を引き継ぐには
データはPC（＝ブラウザ）ごとに保存されるため、ファイルコピーだけでは進捗は移りません。移したいときは:

1. 元のPCで ⚙︎ →「**JSON エクスポート**」（`asayomi_YYYY-MM-DD.json` が保存される）。
2. そのJSONを別PCへ移す。
3. 別PCで ⚙︎ →「**JSON インポート**」でファイルを選ぶ。

> 注意: インポートは既存データを**上書き**します。毎日エクスポートしてバックアップしておくと安全です。
> 両PCで常に同じ進捗を自動同期したい場合は、フェーズ2（バックエンド導入）が必要です（[SPEC.md](SPEC.md) 参照）。
